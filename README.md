# 📊 E-commerce Data Analysis with Python

Este projeto tem como objetivo analisar dados de uma operação de e-commerce para identificar padrões de comportamento, gargalos operacionais e oportunidades de melhoria.

## 🎯 Problema de Negócio
Como entender o comportamento dos clientes, a eficiência logística e o desempenho das categorias de produtos para apoiar decisões estratégicas?

## 🛠️ Tecnologias
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🔍 Principais Análises
- Integração de múltiplas bases de dados (pedidos, clientes, produtos, avaliações)
- Tratamento e limpeza de dados
- Criação de variáveis analíticas (tempo de entrega, valor total)
- Análise de faturamento por categoria e região
- Avaliação de desempenho logístico
- Análise de comportamento de clientes (ativos vs inativos)

## 📈 Insights Gerados
- Alta concentração de receita em poucas categorias
- Diferenças logísticas entre regiões
- Aproximadamente 80% dos clientes inativos
